Active Detection Rules

  • Comment & Structure Analysis: Flags over-explaining ("Step 1:", "Note:"), excessive comments relative to code, and perfect boilerplate formatting.
  • Line Length Variance (Burstiness): Evaluates the standard deviation of line lengths. AI code tends to be highly uniform and visually "blocky".
  • Lexical Entropy: Measures vocabulary variation. Repetitive tokens, generic names (temp, item, val), and repetitive structures decrease entropy.
  • Conversational Leakage: Scans for exact phrases LLMs output when copy-pasted directly (e.g., "Certainly!", "Here is the updated code").
  • Modern Syntax Fingerprinting: Detects excessive modern syntactic sugar.
  • Human Indicators: Detects messy behaviors like multiple comment slashes (////), commented-out code blocks, and aggressive debug statements.
  • Mathematical & Statistical: Analyzes cyclomatic complexity vs line count, operator density, and calculates variable name length variance.
