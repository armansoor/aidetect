# AI Code Analyzer Pro

A 100% local, purely client-side heuristic engine designed to detect code generated by Artificial Intelligence (like ChatGPT, Claude, etc.) versus human-written code. It runs entirely in your browser using JavaScript and does not send any of your code or data to an external API, making it fully secure.

"A Vibe coded website to detect Vibe coded website"

## Features

- **100% Local Execution**: All logic, including Abstract Syntax Tree (AST) profiling and entropy math, runs within your browser.
- **Dark Mode Support**: Built-in toggle for light and dark modes, and inherently respects your system preferences.
- **Drag and Drop / File Upload**: Easily upload files (.js, .py, .html, .css, .json, .ts, .txt, .md) directly for analysis, or paste your code snippet into the textarea.
- **Detailed Flagging System**: The analyzer runs code through various heuristic modules to build a score:
  - **Comment & Structure Analysis:** Checks for over-explanation ("Step 1:", "Note:"), excessive commenting, and AI boilerplate.
  - **Line Length Variance (Burstiness):** Calculates the standard deviation of line lengths. AI code is often highly uniform.
  - **Lexical Entropy:** Checks for repetitive tokens, variable names (e.g., `temp`, `data`), and vocabulary variation.
  - **Conversational Leakage:** Detects if direct LLM phrases were accidentally copy-pasted (e.g., "Certainly!", "Here is the updated code").
  - **Advanced Syntax Overkill:** Flags excessive use of advanced language syntax (like optional chaining) in very small, simple scripts.
  - **Formatting Perfection:** AI rarely leaves messy trailing spaces or inconsistent lines.
  - **Human Indicators:** Detects "messiness" typical of humans, such as commented-out code, aggressive logging (e.g., `console.log('wtf')`), and sloppy comments (`////`).

## How to use

Since it uses vanilla JavaScript and standard HTML/CSS without ES Modules, no server setup is required.

1. Clone or download this repository.
2. Open `index.html` directly in your favorite modern web browser (just double-click the file).
3. Paste code into the input box or drag-and-drop a supported file.
4. Click **Analyze Source**.

## File Structure

- `index.html`: The main user interface, built with HTML and Tailwind CSS (via CDN).
- `style.css`: Custom animations, scrollbar styling, and minor tweaks.
- `analyzer.js`: The core heuristic engine handling math, statistics, and pattern recognition.
- `app.js`: DOM manipulation, drag-and-drop functionality, event listeners, and UI logic.
